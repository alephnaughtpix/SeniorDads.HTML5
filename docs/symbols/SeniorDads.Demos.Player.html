SeniorDads.Demos.Player(container, name, demoClass, width, height)

SeniorDads.Demos.Player

"YouTube" style player for Senior Dads demos

var myPlayer = new SeniorDads.Demos.Player(container, name, demoClass, width, height);

This class allows you to add individual screens (See SeniorDads.Demos.Demo.DemoPart), and to handle general loading of resources, and initialisation before starting the demo. There is also a preloader, which can be used if you need to set something up before loading resources. (eg a "Now loading demo" type screen.)

Once the demo is started, the class can handle moving through the list of screens, or stopping the demo altogether.

Both the preloader and loaders can be supplied with callback functions to be executed when they're done.

Parameters:
{DOM object} container
HTML element from the web page to put the player in
{string} name
Unique name for the demo element
{object} demoClass
Name of the demo class.
{int} width Optional
Width of the player.
{int} height Optional
Height of the player.
