SeniorDads.ScreenHandler(element, name, width, height, bgColour)

SeniorDads.ScreenHandler

{@class}

var myScreenHandler = new SeniorDads.ScreenHandler(element, name, width, height);

This is a screen handler which handles the creation and usage of canvas screens.

From here, individual screens can be added via SeniorDads.ScreenHandler.Screen.

Parameters:
{DOM Object} element
HTML DOM element to place the screens in.
{string} name
Unique name ID for the screen handler.
{number} width
Width of the screen.
{number} height
Height of the screen.
{string} bgColour Optional
Background colour of screen handler.
